My Projects

K2A Fleets and Machineries Inc. - SQL Server Case Study

A SQL-based simulation project developed to test an S&M Software for a heavy equipment supplier. Implemented using normalized relational tables, advanced SQL features (e.g., GROUPING SETS, CUBE, ROLLUP), and a dynamic dataset of 1 million transactions across 5 years. This project includes database modeling, data generation, analytics, and business rule implementation using SQL Server (T-SQL).

Tesca Grocery Chain Project

Designed and implemented an Enterprise Data Warehouse for Tesca Grocery Chain, encompassing ETL pipelines (SSIS) for data automation, data de-duplication, and robust data monitoring (SSMS). Developed both Multidimensional and Tabular models using SQL Server Analysis Services (SSAS) to address complex analytic requirements. Created comprehensive Power BI visualizations to provide actionable insights into sales, promotions, product mix, vendor efficiency, and HR trends (employee misconduct and absence), enabling informed business decisions.

Falanza Agro-Allied Limited Project

Addressed key concerns for Falanza Agro-Allied Limited, including inconsistent egg production, daily mortality, and rising costs, by developing comprehensive Power BI solutions. Replicated existing data sheets and performed in-depth monthly revenue analysis (egg sales, mortality, feeding costs). Conducted trend analysis and forecasting for egg production to advise on bird disposition, analyzed mortality rates based on events, and provided weekly, monthly, and yearly feeding and costing analysis. This enabled data-driven decisions for improved farm management and profitability.

Power BI Report

John Camry Income and Expenditure Analysis – Power BI Dashboard

Designed a Power BI dashboard for a cab owner to monitor and analyze income and expenses. Integrated indicators, trendlines, and forecasting to support cost control and profit visibility. Dashboard is optimized for mobile, tablet, and desktop viewing while retaining Excel-based data entry.
